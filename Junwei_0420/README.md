# GNN

## Pretrain Model
- GCN+GRU
- SingleGCN+GRU


## Pretrain Task (.ipynb)
預訓練任務共有三種，分別是：
- Link Prediction
- Node Classification
    1. label_0
    2. label_mul

## Downstream Task (.ipynb)
下游任務共有五種，分別是：
-  objsum： 消費總金額(回歸)
-  tscnt：交易次數(回歸)
-  spcnt：交易商店種類數量(回歸)
-  label_0：下個月是否消費(分類)
-  label_mul：交易總金額區間(分類)
